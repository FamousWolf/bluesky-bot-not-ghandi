# Bluesky bot "Not Ghandi"

Bluesky bot giving quotes that aren't from Ghandi.

# Requirements

- Python 3.8 or higher
- atproto 0.0.55

# Installation

- Clone the project `git clone git@github.com:FamousWolf/bluesky-bot-not-ghandi.git`
- Install atproto `pip install 'atproto==0.0.55'` (it might with other version, but has only been tested with 0.0.55)
- Copy `config.ini.example` to `config.ini` and add username and password to it
